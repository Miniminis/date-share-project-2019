# DateShareProject
> Html5, Css3, Javascript, JQuery, Jsp, Oracle DB, Ajax 를 이용해 만든 데이트 콘텐츠 공유 플랫폼
<img src="ProjectFlow.png">

## 목차 
* 회원가입/로그인/로그아웃/회원탈퇴
* 마이페이지
* 맛집 게시판
* 엑티비티 게시판
* 영화 게시판
* 쪽지

## 기술 
* `HTML5`
* `CSS3`
* `Javascript`
* `jQuery`
* `JSP` 이용하여 내부 객체 속성으로 데이터 처리, EL + JSTL 
* `JDBC` - `Oracle` + DAO 패턴 이용 
* 커넥션 풀 등록 
* `ajax` 비동기통신
